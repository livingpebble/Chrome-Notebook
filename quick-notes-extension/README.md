# QuickNotes - Chrome 笔记扩展

QuickNotes 是一个简洁实用的 Chrome 浏览器笔记扩展，让你可以随时记录灵感碎片，所有笔记以纯文本格式存储在本地文件夹中。

## 功能特点

- 📝 **简洁文本编辑器** - 专注写作，无干扰界面
- 💾 **自动保存** - 输入后1.5秒自动保存，无需手动操作
- 📂 **本地存储** - 笔记存储为 txt 文件，可直接访问
- 🔍 **快速搜索** - 通过文件名搜索笔记
- 📋 **历史记录** - 查看所有笔记列表，按修改时间排序
- 🗑️ **删除管理** - 轻松删除不需要的笔记
- ⏰ **时间显示** - 显示每条笔记的最后修改时间

## 安装步骤

### 1. 下载扩展文件
将 `quick-notes-extension` 文件夹复制到本地任意位置（建议放在用户目录下）

### 2. 在 Chrome 中加载扩展
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `quick-notes-extension` 文件夹

### 3. 首次使用设置
1. 点击 Chrome 工具栏的扩展图标
2. 首次使用会弹出目录选择框
3. 选择一个文件夹作为笔记存储目录
   - 推荐创建一个专门的文件夹（如 `MyNotes`）
   - 建议将扩展文件夹和笔记文件夹分开管理

## 使用指南

### 创建新笔记
1. 点击工具栏的 **"新建"** 按钮
2. 输入文件名（无需输入 .txt 后缀）
3. 点击 **"创建"** 开始编辑

### 编辑笔记
- 直接在文本区域输入内容
- 输入后1.5秒自动保存
- 也可以点击 **"保存"** 按钮手动保存

### 查看历史笔记
- 左侧边栏显示所有笔记列表
- 按修改时间降序排列
- 点击任意笔记打开

### 搜索笔记
- 在搜索框输入文件名关键词
- 实时过滤显示匹配的笔记

### 删除笔记
1. 打开要删除的笔记
2. 点击工具栏的 **"删除"** 按钮
3. 确认删除操作

## 技术说明

- **Manifest Version**: 3
- **存储方式**: Chrome FileSystem API
- **文件格式**: UTF-8 编码的纯文本文件（.txt）
- **自动保存**: 1500ms 延迟防抖

## 注意事项

1. **首次授权**: Chrome 会要求授权文件系统访问，请允许
2. **数据安全**: 笔记直接存储为 txt 文件，可使用任何文本编辑器打开
3. **备份建议**: 定期备份 notebooks 文件夹
4. **文件命名**: 避免使用特殊字符作为文件名

## 常见问题

### Q: 笔记存储在哪里？
笔记存储在你选择的文件夹中（首次使用时指定），每个笔记都是一个独立的 .txt 文件。

### Q: 如何备份笔记？
只需备份你选择的笔记文件夹即可，所有笔记都是标准格式的 txt 文件。

### Q: 能否在不同电脑间同步？
可以将笔记文件夹放在云同步服务（如 Dropbox、OneDrive）中实现跨设备同步。

### Q: 扩展图标显示不正确？
可以使用图标生成工具重新生成图标文件，或使用系统自带图标。
