# QuickNotes - Chrome 笔记扩展

QuickNotes 是一个简洁实用的 Chrome 浏览器笔记扩展，让你可以随时记录灵感碎片，所有笔记安全存储在浏览器本地存储中。

## 功能特点

- 📝 **简洁文本编辑器** - 专注写作，无干扰界面
- 💾 **自动保存** - 输入后1.5秒自动保存，无需手动操作
- 💽 **本地存储** - 笔记安全存储在浏览器本地存储中
- 🔍 **快速搜索** - 通过标题搜索笔记
- 📋 **历史记录** - 查看所有笔记列表，按修改时间排序
- 🗑️ **删除管理** - 轻松删除不需要的笔记
- ⏰ **时间显示** - 显示每条笔记的最后修改时间

## 安装步骤

### 1. 下载扩展文件
将 `quick-notes-extension` 文件夹复制到本地任意位置

### 2. 在 Chrome 中加载扩展
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `quick-notes-extension` 文件夹

### 3. 开始使用
1. 点击 Chrome 工具栏的扩展图标
2. 直接点击"新建"按钮创建第一个笔记

## 使用指南

### 创建新笔记
1. 点击工具栏的 **"新建"** 按钮
2. 输入笔记标题
3. 点击 **"创建"** 开始编辑

### 编辑笔记
- 直接在文本区域输入内容
- 输入后1.5秒自动保存
- 也可以点击 **"保存"** 按钮手动保存
- 双击标题可修改笔记名称

### 查看历史笔记
- 左侧边栏显示所有笔记列表
- 按修改时间降序排列
- 点击任意笔记打开

### 搜索笔记
- 在搜索框输入标题关键词
- 实时过滤显示匹配的笔记

### 删除笔记
1. 打开要删除的笔记
2. 点击工具栏的 **"删除"** 按钮
3. 确认删除操作

## 技术说明

- **Manifest Version**: 3
- **存储方式**: Chrome Storage API (localStorage)
- **自动保存**: 1500ms 延迟防抖

## 注意事项

1. **数据安全**: 笔记存储在浏览器本地存储中，清除浏览器数据会导致笔记丢失
2. **备份建议**: 建议定期导出笔记内容进行备份
3. **标题限制**: 避免使用特殊字符作为笔记标题

## 常见问题

### Q: 笔记存储在哪里？
笔记存储在浏览器的本地存储中，每个扩展独立存储。

### Q: 如何备份笔记？
可以复制笔记内容到其他地方进行备份。

### Q: 能否在不同电脑间同步？
当前版本不支持跨设备同步，建议使用云笔记服务进行跨设备管理。

### Q: 扩展图标显示不正确？
可以使用图标生成工具重新生成图标文件，或使用系统自带图标。
